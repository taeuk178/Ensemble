# 로컬 메모 기능 스펙

## 목표

사용자가 텍스트 메모를 로컬에 만들고 삭제할 수 있어야 한다.

## 범위

- 메모 생성, 목록 조회, 삭제를 포함한다.
- 계정과 네트워크 동기화는 제외한다.

## 생성 흐름

1. 사용자가 비어 있지 않은 본문을 입력한다.
2. 저장하면 UUID와 생성 시각을 포함한 메모를 로컬 저장소에 기록한다.
3. 저장 실패 시 입력을 유지하고 오류를 표시한다.

## 삭제 흐름

1. 삭제 선택 시 확인 화면을 표시한다.
2. 확인하면 로컬 저장소에서 삭제한다.
3. 삭제 실패 시 기존 목록을 유지하고 오류를 표시한다.

## 완료 조건

- 저장 후 동일한 본문이 목록에 표시된다.
- 저장 실패를 주입하면 입력과 오류가 함께 표시된다.
- 삭제 성공 후 목록에서 메모가 사라진다.
- 삭제 실패를 주입하면 메모가 목록에 남는다.
