# Fixture: Clean specification

## 목표

사용자가 로컬 메모를 만들고 삭제할 수 있어야 한다.

## 범위

- 텍스트 메모 생성
- 메모 목록 조회
- 메모 삭제
- 네트워크 동기화와 계정 기능은 제외

## 흐름

### 생성

1. 사용자가 비어 있지 않은 본문을 입력한다.
2. 저장을 누르면 UUID와 생성 시각을 포함한 메모가 로컬 저장소에 기록된다.
3. 저장 실패 시 입력을 유지하고 오류를 표시한다.

### 삭제

1. 사용자가 삭제를 선택하면 확인 화면을 표시한다.
2. 확인하면 로컬 저장소에서 삭제한다.
3. 삭제 실패 시 목록을 유지하고 오류를 표시한다.

## 완료 조건

- 비어 있지 않은 본문을 저장하면 목록에 동일한 본문이 표시된다.
- 저장소 오류를 주입하면 입력이 유지되고 오류가 표시된다.
- 삭제 확인 후 목록에서 메모가 사라진다.
- 삭제 오류를 주입하면 메모가 목록에 남고 오류가 표시된다.
